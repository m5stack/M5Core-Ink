## Description

**CoreInk** is a brand new E-ink display in the M5Stack cores range. Controlled by the ESP32-PICO-D4 This new device includes a 200x200 1.54" Black and White E-Ink Display. Compared to a regular LCDÔºåE-ink displays are easier on the eyes, which makes them a great choice for reading or viewing for longer periods.
